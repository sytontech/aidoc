# 超全项目合集

构建数字人的思路可以分为三步：
- 构铸血肉 —— 创建数字人形象
- 创造感官 —— 文本转音频，音频与形象结合让数字人“开口说话”
- 注入灵魂 —— 输入领域知识，实现智能对话

因此我们根据不同的领域整理了众多优秀项目，总有一款适合你！
