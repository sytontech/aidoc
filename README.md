## 安装指南

本项目基于Meta 开源项目 Docusaurus 。
安装示例：
npx create-docusaurus@latest my-website classic

运行：
npm run start

Build:
npm run build

更新：
npm install

要检查更新是否成功，可以运行：

npx docusaurus --version

安装指南参考：
[Docusaurus] (https://docusaurus.io/zh-CN/docs/installation)
